The project implements the in mempru database. It is used for managing the values of database variables. The project in implemented in JAVA.
 
 Compiling the code:
 
 javac Variable.java
 javac Database.java
 javac RunDB.java
 
 Running the database
 
 java RunDB -i input-file
 
 input-files are:
 
 input1.txt
 input2.txt
 input3.txt
 input4.txt
